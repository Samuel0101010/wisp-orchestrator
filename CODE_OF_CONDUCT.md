# Code of Conduct

WISP adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its Code of Conduct. All contributors, maintainers, and community members are
expected to follow its terms when participating in this project — including issues,
pull requests, discussions, reviews, and any other project space.

The full canonical text is maintained at:

- HTML: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Plain text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct.txt>

By contributing to WISP you agree to abide by that document.

## Reporting concerns

If you believe someone has not followed the Code of Conduct, please report it
privately through one of the following channels:

1. **Preferred** — open a private
   [GitHub Security Advisory](https://github.com/Samuel0101010/wisp-orchestrator/security/advisories/new)
   and note in the title that it is a community-conduct report.
2. Contact the project maintainers via a direct GitHub message.

All reports are reviewed promptly and handled confidentially. Maintainers will
acknowledge receipt within seven days and coordinate any follow-up with the
reporter.

## Enforcement

Enforcement follows the four-step ladder defined in the Contributor Covenant 2.1
Enforcement Guidelines (correction, warning, temporary ban, permanent ban). The
maintainer team is the final authority on enforcement decisions for this project.
