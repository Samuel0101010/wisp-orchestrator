# WISP v1.0 — One-Page Summary

Tagged `v1.0.0` on commit `928b194` (2026-05-06). Delivered in 8 PRs (#3–#10) shipped over a single concentrated session against the v1.0 plan written 2026-05-05. Detailed run-by-run findings are in [`real-run-notes.md`](./real-run-notes.md) (~700 lines); this file is the index.

## What v1.0 is

A local-first orchestrator for Claude Code that runs goal-to-PR loops with 1–8 named roles, shared per-run memory across tasks, built-in templates, automatic QA-replan with audit trail, and slash-command shortcuts. Driven against real Claude Max in 7 R-validation runs.

## Architectural layering (M1 → v1.0)

The M1 vertical-slice core (`Walker` / `SubprocessPool` / `Worktree` / `Verification`) was preserved unchanged. v1.0 layered on top via a single seam each:

| Layer | Seam | What v1.0 added |
|---|---|---|
| Schema | `teamSchema`, `successCriteriaSchema`, `plans` table | `{roles: AgentSpec[]}`, `preflight`, `parent_plan_id` |
| Walker | `WalkerDeps` interface | `replanOnQAFailure?` callback, branch-prefix v\<N> for replans |
| Subprocess | `SubprocessPool.defaultMcpConfigPath` | per-run memory MCP injection via `--mcp-config` |
| Server | `RunRuntime.makeWalkerDeps` | wires replan helper + memory-config writer |
| Events | `HarnessEvent` discriminated union | `harness.verify-failed`, `qa.replan-triggered`, `qa.replan-exhausted` |
| HTTP | `routes/index.ts` | `/api/team-templates`, `/api/plans/:id/chain`, `/api/runs/:id/events` |
| UI | `TeamBuilder`, `PlanEditor`, `Sidebar` | TeamRoleCard add/remove, TemplatePicker, PlanVersionBadge |
| Plugin | `.claude-plugin/plugin.json` | `skills` directory entry + 4 SKILL.md files |

## Known limitations preserved into v1.0

- **Planner-event audit gap.** Planner runs broadcast on `planner:<projectId>` WS but aren't in `events` (no `runId` parent).
- **Replan task-totals.** `tasks` table has one row per task id; v2 plan reusing a task id overwrites v1's tokens. `events` has full history.
- **Single-tenant.** SQLite + per-machine paths assume one user.
- **No marketplace publication.** Plugin stays private.
